# core-mvc-column-define-validate
[ASP.NET Core MVC][Vue3] 統一欄位格式定義及驗證設計範例

在開發網站系統設計階段，對於欄位格式定義 (Data Annotation) 及驗證 (Validate) 就要先設計好，讓相同欄位在不同頁面使用時，都能維持相同的格式驗證，當欄位格式有異動時，也只需要修改一個地方就好，讓所有使用到的頁面都能一致同步修改。
